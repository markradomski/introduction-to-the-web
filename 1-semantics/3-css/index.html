Cascading Style Sheets

Holy shit, we are not looking at that eye-straining browser default design anymore, you could almost say it looks nice. We altered the look & feel of our document by using CSS!

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language. (from Wikipedia).
Let's translate this to human - CSS is a language used to visually enhance the presentation of your HTML document.

How does it work?

CSS is very simple! and it is loads of fun to play around with it.

  1. Create a new folder on a location you will remember, perhaps (dare I say it) Desktop?, folder-name does not matter! If you really can't decide on a folder-name, then lets call it xx-my_awesome_project-xx

  2. Copy semantics.html from the previous chapter and paste the file into your newly-made folder!

  3. Create a new file in your directory, call it my-style.css. Your folder-structure should now look like the following:

    YourNewlyCreatedFolder
     |-semantics.html
     |-my-style.css

  4. Within my-style.css type the following: body { background-color: red; }. We have written our first styles!

  5. Refresh your HTML file, you will notice the background is... not red

  6. We still need to "link" the file! Luckily the <head> can request this for us! In order to request the file, put the following in your document's <head>: <link rel="stylesheet" href="my-style.css">
